¿Qué es?

El modelo Open Systems Interconnection (OSI) es un modelo conceptual creado por la Organización Internacional para la Estandarización, el cual permite que diversos sistemas de comunicación se conecten usando protocolos estándar. En otras palabras, el OSI proporciona un estándar para que distintos sistemas de equipos puedan comunicarse entre sí.

Una arquitectura en capas que define una estructura estandarizada para la interconexión de sistemas de comunicaciones. Cada capa tiene un propósito específico, y las capas superiores dependen de los servicios proporcionados por las capas inferiores para funcionar correctamente.
Tanenbaum, A. S. (2003). Redes de computadoras (4ª ed.). Prentice Hall.

Capas del Modelo OSI

El modelo OSI esta conformado por 7 capas las cuales son:

  • Capa Fisica
  • Capa de Enlace de Datos
  • Capa de Red
  • Capa de Transporte
  • Capa de Sesión
  • Capa de Presentación
  • Capa de Aplicación
